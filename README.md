# 🕵️ UNDERCOVER — Party Word Game

A fully interactive, dark-themed, mobile-friendly party word game inspired by *Undercover*.

## 🎮 How to Play
- **Civilians** share one secret word
- **Undercover** gets a similar but different word  
- **Mr. White** gets no word — must bluff!
- Give clues, vote out the suspicious player
- Civilians win if all undercovers are caught
- Undercovers win if they match civilian count
- Mr. White can guess the civilian word to steal the win!

## ✨ Features
- 200+ curated word pairs (landmarks, food, tech, fashion, nature, psychology & more)
- Pass-the-phone secret role reveal system
- Optional discussion countdown timer
- Voting & elimination with reveal animations
- Persistent scoreboard across rounds
- Web Audio sound effects
- Fully responsive — works great on mobile
